## 自定义资产

该项目包含可以嵌入 Carla 项目中的自定义资产，比如：车辆、行人等。

### 使用
直接覆盖到carla目录下，然后运行脚本进行测试：
```python
python manual_control.py --filter vehicle.BYD.seal
```
